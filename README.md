Your teaching assistant for Fusion.

## üöÄ Quick Start

1. Download the add-in package `\FusBuddy360`
1. Install via Fusion 360 Add-Ins menu
1. Add in your Gemini API key
1. Start modeling

How to install a Add-In:

1. Go to **Utilities > Add-Ins**
1. Click the **"+"** button to add a new add-in
1. Select your add-in folder (e.g., `FusBuddy360`)
1. Find your add-in in the list and click **Run**
